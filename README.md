# Sudoku Solver (Python, OOP, PEP 8)

یک حل‌کنندهٔ سودوکو با طراحی شی‌ءگرا (OOP) و رعایت استاندارد PEP 8.

## ساختار پروژه

- `sudoku.py` — کلاس اصلی `Sudoku` و توابع کمکی برای:
  - خواندن جدول‌ها از فایل `sudokus.txt`
  - حل هر سودوکو با بک‌ترکینگ (backtracking)
  - محاسبهٔ عدد سه‌رقمی اول هر جواب و جمع‌زدن آن‌ها
- `LICENSE` — لایسنس MIT برای استفاده، ویرایش و انتشار آزاد کد.
- `sudokus.txt` — نمونه‌ی فایل ورودی (هر جدول با خط `sdk` شروع می‌شود).

## فرمت فایل ورودی (`sudokus.txt`)

هر سودوکو این‌طوری تعریف می‌شود:

```text
sdk
003020600
900305001
001806400
008102900
700000008
006708200
002609500
800203009
005010300
```

- خطی که با `sdk` شروع می‌شود یعنی شروع یک جدول جدید.
- بعد از آن دقیقاً **۹ خط** عددی داریم.
- عدد `0` یعنی خانهٔ خالی.

می‌توانید چندین جدول پشت سر هم داشته باشید:

```text
sdk
...
9 line grid
sdk
...
9 line grid
```

## اجرای برنامه

داخل پوشه‌ای که فایل‌های زیر را دارید:

- `sudoku.py`
- `sudokus.txt`
- `LICENSE`

این دستور را اجرا کنید:

```bash
python sudoku.py
```

### خروجی

برای هر سودوکو:

- در کنسول مراحل حل نمایش داده می‌شود (جدول حل‌شده)
- اگر جدول غیرقابل حل باشد، پیام `has no solution. Skipping.` نمایش داده می‌شود.

در انتها، مجموع عدد سه‌رقمی اول هر جدول حل‌شده چاپ می‌شود، مثلاً:

```text
Total sum of first 3 digits of each solution: 728
```

## استفاده به صورت ماژول

می‌توانید این پروژه را به‌صورت ماژول هم استفاده کنید:

```python
from pathlib import Path
from sudoku import Sudoku, parse_sudoku_file

for puzzle in parse_sudoku_file(Path("sudokus.txt")):
    if puzzle.solve():
        print(puzzle)
```
